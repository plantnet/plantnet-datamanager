* New synchronization wizard - step 3/4

One-shot: The synchronization will be executed once, and stopped as soon as it has finished processing all the documents.

Continuous: The synchronization will be executed continuously to keep the two databases in the same state at all times. Beware that document deletions will be reproduced as well, without warning.

{{#hasFilter}}

Without deleted docs: The documents that were deleted won't be replicated. Only docs that were modified or newly created docs will be synchronized.

With deleted docs: The documents that were deleted will be replicated as well. Beware that if you chose to synchronize certain structures only, deleted docs from other structures will be replicated as well: this may lead to unexpected results!

{{/hasFilter}}
